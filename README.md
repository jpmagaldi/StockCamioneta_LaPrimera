# StockCamioneta_LaPrimera

(This product was designed for a Spanish-speaking company)

Initial project for online stock products in a transport truck products via mobile device.

This app is designed for administrators to use. It is not a client-facing app or employee-facing app.

The initial idea is to use the same database currently used by the company. Therefore, a file called Servidor.js is provided, which acts as a "server" to handle requests for generating queries to the database and others backend stuff.

You just need to modify some company-specific details in the Servidor.js file and then run it with python.


### Features:

- Stock Management:
    - CRUD for products in the truck at the start, end and during the truck's route

- Sales:
    - See all products in differents tables
        - Initial stock
        - Final stock
        - Products sold
        - Products returned

- Alerts and errors when there is no connection with the server
